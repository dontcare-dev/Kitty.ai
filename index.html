Common Questions

FAQ

How do daily limits work? ▼
Daily limits reset every 24 hours from your first request of the day. You can switch between models freely — each model has its own separate limit. Unused requests don't carry over.
Which model should I use? ▼
For quick tasks, casual conversation, and brainstorming, use Gemini 3.1 Lite (100/day). For code generation, detailed writing, and analysis, use Gemini 3.5 (50/day). For the most complex reasoning, research, and expert-level tasks, use Gemini 3.1 Pro (30/day).
Is my data private? ▼
All conversations are stored locally in your browser. We implement XSS protection and input sanitization. For maximum privacy, use the export feature and clear history regularly.
Can I export my conversations? ▼
Yes! Go to Settings → Export Conversations to download all your chat history as a JSON file. You can re-import this data anytime.
How do I access the admin panel? ▼
The admin panel is accessible only to accounts with admin role. Register with username "admin" to get admin access in the demo. From dashboard, navigate to Admin Panel in the sidebar.
